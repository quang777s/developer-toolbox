# Voice 3 Sound Files

This directory should contain the sound files for voice 3.

## Required Files

Add the following MP3 files:

### Numbers (1-90)
- `1.mp3`, `2.mp3`, `3.mp3`, ... `90.mp3`

### Special Sounds
- `start.mp3` - "1, 2, 3 bắt đầu" announcement
- `funny-45.mp3` - Plays at 45 numbers called
- `funny-60.mp3` - Plays at 60 numbers called
- `funny-75.mp3` - Plays at 75 numbers called
- `funny-90.mp3` - Plays at 90 numbers called

## File Format
- Format: MP3 recommended (or WAV, OGG, M4A)
- Duration: 1-3 seconds per file
- Volume: Normalized level
